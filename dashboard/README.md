## Estrutura do dashboard

O painel foi organizado em cinco perspectivas principais:

1. Panorama geral do acervo  
2. Aquisições ao longo do tempo  
3. Estrutura da coleção  
4. Distribuição por década de criação  
5. Representatividade dos artistas  

O objetivo do dashboard é complementar a análise exploratória, oferecendo uma leitura visual e integrada dos principais padrões identificados no dataset.
